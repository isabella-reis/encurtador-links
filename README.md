# 🔗 Brev.ly — Encurtador de Links

**Brev.ly** é um encurtador de links full stack desenvolvido como parte de um desafio da pós-graduação.  
O projeto permite a criação de URLs curtas personalizadas, redirecionamento automático, contagem de acessos e exportação em CSV.
